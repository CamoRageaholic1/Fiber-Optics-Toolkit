# Examples

Python code examples for the Fiber Optics Toolkit.

## Link Budget Examples

See `example_link_budget.py` for various scenarios:
- Basic 10km singlemode link
- Long-haul 50km link
- Data center multimode link
- Marginal link analysis

## Usage

```bash
python examples/example_link_budget.py
```
